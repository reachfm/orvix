## RC1 → RC2 Changes

### Critical Fixes

1. **SQLite CGO Issue** (CRITICAL)
   - **Problem**: Binary compiled with `CGO_ENABLED=0` failed on SQLite
   - **Root Cause**: `mattn/go-sqlite3` requires CGO
   - **Solution**: Replaced with `modernc.org/sqlite` (pure Go)
   - **Files**: `go.mod`, `internal/config/database.go`

2. **Stalwart Download 404** (CRITICAL)
   - **Problem**: GitHub release URL returned 404
   - **Root Cause**: Incorrect URL format for Stalwart releases
   - **Solution**: Use stalw.art API with fallback to GitHub
   - **Files**: `release/install.sh`

3. **systemd Warning** (WARNING)
   - **Problem**: Unknown key `StartLimitIntervalSec`
   - **Root Cause**: Invalid systemd directive for Ubuntu 22.04+
   - **Solution**: Removed directive, simplified service file
   - **Files**: `release/systemd/orvix.service`

4. **Installer UX Issues** (MEDIUM)
   - **Problem**: No input validation, confusing prompts
   - **Solution**: Added validation loops, better error messages
   - **Files**: `release/install.sh`
