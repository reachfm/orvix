# Security Settings

This page covers the security features available for your Orvix account.

## Multi-Factor Authentication (MFA)

MFA adds an extra layer of protection to your account. See the [MFA setup guide](mfa.md) for step-by-step instructions.

## Session Management

You can view and manage your active sessions:

1. Go to **Account Settings** > **Security** > **Sessions**.
2. Review the list of currently active sessions showing the device, IP address, and last activity time.
3. Click **Revoke** next to any session you do not recognize or no longer need.

## Login History

1. Go to **Account Settings** > **Security** > **Login History**.
2. View recent sign-in attempts, including IP address, location, and whether each attempt succeeded or failed.
3. If you see any unrecognized activity, change your password immediately and revoke all sessions.

## Organization Security Settings

Organization admins can enforce security policies for all members:

| Setting | Description |
| ------- | ----------- |
| **Require MFA** | All members must enable multi-factor authentication. |
| **Session Timeout** | Automatically sign out idle users after a configurable period. |
| **IP Allowlisting** | Restrict access to specific IP addresses or ranges. |

Configure these under **Organization** > **Settings** > **Security**.

## Reporting a Security Issue

If you believe your account has been compromised or you have found a security vulnerability, contact security@orvix.com immediately.
