# Using Orvix Webmail

Orvix Webmail allows you to access your email from any browser without installing software.

## Signing In

1. Go to [https://webmail.orvix.com](https://webmail.orvix.com).
2. Enter your full email address and password.
3. Click **Sign In**.

If MFA is enabled on your account, enter the one-time code when prompted.

## Interface Overview

| Section | Description |
| ------- | ----------- |
| **Inbox** | Your primary incoming mail folder. |
| **Sent** | Copies of messages you have sent. |
| **Drafts** | Messages you started but have not sent. |
| **Trash** | Deleted messages (emptied after 30 days). |
| **Spam** | Filtered spam messages. |
| **Folders** | Custom folders you create for organization. |

## Composing an Email

1. Click **Compose** in the top-left corner.
2. Fill in the **To**, **Cc**, and **Bcc** fields.
3. Enter a **Subject**.
4. Write your message in the body area.
5. To attach files, click the paperclip icon or drag files into the compose window.
6. Click **Send**.

## Replying and Forwarding

- **Reply**: Opens a compose window addressed to the sender only.
- **Reply All**: Opens a compose window addressed to the sender and all other recipients.
- **Forward**: Opens a compose window with the message content quoted. Add the recipient and click **Send**.

## Organizing with Folders

1. In the left sidebar, click **New Folder** (or the `+` icon).
2. Enter a folder name (e.g., `Invoices`, `Projects`).
3. Click **Create**.
4. Drag messages into the folder to organize them.

## Search

Use the search bar at the top to find messages by:

- Sender or recipient
- Subject keywords
- Body content

Use advanced search operators for precise results:

- `from:john` — messages from a specific sender
- `to:jane` — messages to a specific recipient
- `subject:invoice` — messages with a keyword in the subject
- `has:attachment` — messages with attachments
- `before:2026-01-01` / `after:2025-12-31` — date ranges

## Keyboard Shortcuts

| Shortcut | Action |
| -------- | ------ |
| `c` | Compose new message |
| `r` | Reply |
| `a` | Reply all |
| `f` | Forward |
| `Ctrl + Enter` | Send message |
| `/` | Focus search bar |
| `j` / `k` | Next / previous message |

- [Vacation auto-replies](vacation.md)
