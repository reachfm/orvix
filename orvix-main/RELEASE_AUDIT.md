# Orvix Release Audit

## Required Runtime Files
- `orvix-v1.0.4-linux-amd64` — Built Go binary for Linux AMD64 (SHA256: e7ad824523dea77858b11dfcc06793bb868a1141bf1f95dd9f511b4317b1138b)
- `orvix.yaml` — Configuration file (generated by installer)
- `stalwart` — Stalwart mail server binary v0.16.7 (downloaded by installer)

## RC5 Runtime Fixes
- **Systemd ReadWritePaths**: Added for /etc/orvix, /var/lib/orvix, /var/log/orvix
- **Stalwart config.json**: v0.16.7 uses JSON format, data path inside config file
- **Redis**: Now installed and enabled by installer

## RC3 Security Requirements
- Admin credentials MUST be provided via environment variables:
  - `ORVIX_ADMIN_EMAIL`
  - `ORVIX_ADMIN_PASSWORD`
- NO default credentials in source code

## Required Directories
| Path | Purpose | Created by |

| `/etc/orvix/` | Configuration files | Installer |
