# Orvix Release Audit

## Required Runtime Files
- `orvix-linux-amd64` — Built Go binary for Linux AMD64
- `orvix.yaml` — Configuration file (generated by installer)
- `stalwart` — Stalwart mail server binary (downloaded by installer)

## Required Directories
| Path | Purpose | Created by |
|------|---------|------------|
| `/etc/orvix/` | Configuration files | Installer |
| `/var/lib/orvix/` | Runtime data (DB, certs) | Installer |
| `/var/lib/orvix/stalwart/` | Stalwart data | Installer |
| `/var/log/orvix/` | Log files | Installer |
| `/usr/local/bin/` | Orvix binary | Installer |

## Required Services
| Service | Purpose | Managed by |
|---------|---------|------------|
| `orvix.service` | Orvix server (starts Stalwart) | systemd |
| `orvix-webmail.service` | (optional) Dev server | systemd |

## Required Ports
| Port | Service | Configurable |
|------|---------|-------------|
| 8080 | Admin API (Orvix) | Yes |
| 80 | HTTP redirect (optional) | No |
| 25 | SMTP (Stalwart) | Stalwart config |
| 143, 993 | IMAP (Stalwart) | Stalwart config |
| 110, 995 | POP3 (Stalwart) | Stalwart config |
| 18080 | Stalwart API | Stalwart config |

## Required Dependencies
| Dependency | Version | Purpose |
|------------|---------|---------|
| Go (build only) | 1.25+ | Build Orvix |
| Node.js (build only) | 20+ | Build frontend |
| SQLite 3 | 3.x | Default database |
| Redis | 7+ | Caching/rate limiting (optional) |
| PostgreSQL | 16+ | Production database (optional) |
| Stalwart | 0.10.x | Mail engine (downloaded by installer) |

## Release Package Structure
```
release/
├── install.sh              # Production installer
├── uninstall.sh            # Clean uninstall
├── upgrade.sh              # Version upgrade
├── checksums.txt           # SHA256 checksums
├── RELEASE_NOTES.md        # Version changelog
├── VERSION                 # Version file
├── orvix-linux-amd64       # Go binary (Linux AMD64)
├── systemd/
│   └── orvix.service       # systemd service unit
├── configs/
│   └── orvix.yaml.example  # Example configuration
└── scripts/
    ├── healthcheck.sh      # Health check script
    └── diagnostics.sh      # Diagnostics script
```
